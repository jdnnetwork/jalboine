## 이번에 만들 것 (Phase 1만)

### 1. 첫 화면
- 상단: "잘보이네" 로고
- 가운데 크게: "시작하기" 버튼 (화면 절반 이상)
- 버튼 아래: "(이 버튼을 누르세요)"
- 하단 작은 박스: "가족 및 어르신을 도와주시는 분은 여기를 눌러주세요"
- "시작하기" → 기기 ID로 익명 계정 자동 생성 → 연령대 선택으로
- "가족 도와주시는 분" → 보호자 로그인으로

### 2. 연령대 선택
- "연세가 어떻게 되세요?" + 음성 재생 (age.mp3)
- 큰 버튼 5개: 60~64 / 65~69 / 70~74 / 75~79 / 80세 이상

### 3. 온보딩 질문
- 한 화면에 질문 1개 + "네"/"아니요" 큰 버튼
- 음성 파일 자동 재생
- 음성 인식: "네","응","예","그래" → 예 / "아니","아니요" → 아니요
- 하단 진행 표시
- 질문 목록:
  1. 전화 (phone.mp3)
  2. 카카오톡 (kakao.mp3)
  3. 동영상 (youtube.mp3)
  4. 카메라 (camera.mp3)
  5. 사진앨범 (album.mp3)
  6. 약 복용 (medicine.mp3) → "네" 시 약 설정

### 4. 약 복용 설정
- "하루에 몇 번?" → 1/2/3 큰 버튼 (how_many.mp3)
- "몇 시?" → 큰 숫자 버튼 6~12시 (what_hour.mp3)
- Supabase medications 테이블 저장

### 5. 가족 연결 분기
- 딥링크로 설치한 경우 → 자동 연결 → "가족에게 연락" 활성화
- 직접 설치한 경우 → 6자리 코드 생성 → "자녀분에게 이 번호를 알려주세요: 123456" 큰 글씨 + 음성
- "아니요" → 가족 기능 없이 홈 화면 완성

### 6. 메인 홈 화면
- 중앙: 선택한 앱들 큰 카드 (최대 8개, 화면 꽉 채움)
- 우측 상단: 소리 버튼 (소리→진동→무음 전환)
- 우측 하단: "원래 화면" 작은 버튼 (확인 2번)
- 하단 고정: SOS 긴급 전화 빨간 버튼
- 버튼 터치 시 SystemSound.click + HapticFeedback

### 7. 긴급 전화
- 119 (빨간) — "아플 때, 다쳤을 때"
- 112 (파란) — "도움이 필요할 때"
- 보호자 (초록) — 이름 + 번호
- 원터치 전화 연결

### 8. 약 알림
- 설정 시간에 잠금화면 위 팝업
- "약 드실 시간이에요!" + 음성 (medicine_alarm.mp3)
- "먹었어요" / "나중에" 버튼
- "나중에" → 10분 후 재알림
- Supabase med_logs 기록

### 9. 보호자 로그인
- 구글 로그인 버튼
- 카카오 로그인 버튼
- Supabase Auth

### 10. 보호자 — 부모님 연결
- 부모님 전화번호 입력
- "문자 보내기" → 기본 문자 앱 열림 (설치 링크 + 연결 코드 자동 채워짐)
- 딥링크: https://jalboine.app/connect?code=ABC123

### 11. 보호자 — 대시보드 (기본)
- 부모님 폰 상태: 배터리, 소리모드, 인터넷 연결
- 홈 화면 관리: 앱 추가/삭제/순서 변경 (최대 8개)
- 약 관리: 약 추가/수정/삭제, 복용 기록 확인
- 긴급 연락처 설정

## 디자인 스타일

### 피보호자 화면
- 배경: 연한 웜톤 베이지 + 살짝 글래스모피즘 (배경만)
- 카드: 흰색 베이스 + 한쪽 모서리로 색 번지는 그라데이션
- 아이콘/텍스트 진하고 크게 (버튼 24pt 이상)
- 둥근 모서리 (20px+) + 부드러운 그림자
- 파란 계열 피하기
- 버튼 터치 시 클릭 사운드 + 진동

### 보호자 화면
- 흰색 배경 + 파란색 포인트 컬러
- 글래스모피즘 적용 (반투명 카드 + 블러)
- 둥근 흰색 카드 + 은은한 그림자
- 모던하고 깔끔한 UI
